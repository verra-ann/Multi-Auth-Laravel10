# Multi - Authentication System

# Technologies Used
Project is created with :
* Laragon
* Git
* Visual Studio Code
* Cloud (GitHub)
* PHP 8.1.4
* Laravel 10
* Laravel Breeze
* Composer
* Tailwind CSS

# Features
* Allows different roles of users to access in the system 
* Up to three different roles (Super Admin, Admin, Client)
